# SaaS Capacity Planning Project

## Deskripsi Proyek
Aplikasi SaaS ini dirancang untuk membantu tim IT dan QA dalam melakukan capacity planning. Fokusnya adalah pada pengelolaan beban kerja pengujian, pemantauan sumber daya, dan prediksi kebutuhan kapasitas agar proses pengujian berjalan lebih efisien dan efektif.

## Status 
Aplikasi ini sudah dideploy ke production dan dapat diakses di:

https://qapacityplan.netlify.app/

## Tujuan Utama
- Membantu tim IT merencanakan kapasitas aktivitas pengujian.
- Mengoptimalkan alokasi sumber daya untuk tim QA.
- Menyediakan analisis untuk memprediksi kebutuhan kapasitas di masa depan.
- Meningkatkan produktivitas dan kualitas proses pengujian perangkat lunak.

## Fitur Utama
- Dashboard pemantauan kapasitas tim QA.
- Perencanaan beban kerja pengujian.
- Integrasi dengan tools pengujian populer seperti JIRA, TestRail, dan Selenium.
- Laporan otomatis untuk analisis performa dan kapasitas.

## Teknologi yang Digunakan
- Backend: Node.js / Python (Django atau Flask)
- Frontend: React.js / Vue.js
- Database: PostgreSQL / MongoDB
- Deployment: Netlify (production), AWS / Heroku (opsional)

## Struktur Proyek
- `/src`: Kode sumber aplikasi
- `/docs`: Dokumentasi, hasil ide, dan diskusi
- `/tests`: Kode pengujian
- `/config`: Konfigurasi aplikasi
