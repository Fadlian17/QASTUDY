Atlassian Jira

Project Management & Issue Tracking
🔐 Authentication: SSO (SAML/OAuth), 2FA, API tokens, password-less login (Passkeys)
👥 Roles: Admin, Project Manager, Developer, QA, Reporter, Viewer
📊 Dashboard: Customizable widgets, issue filters, sprint board, activity feed
📱 Features: Multi-tenant Mobile app Webhooks REST API

GitHub

Version Control & Collaboration
🔐 Authentication: OAuth, 2FA, SSH keys, GitHub App tokens, Passkeys
👥 Roles: Owner, Admin, Maintainer, Developer, Triager, Viewer
📊 Dashboard: Personalized feed, repos, issues, PRs, projects, discussions
📱 Features: Code review CI/CD Security scanning Automation
🔗 Reference: github.com

Slack

Team Communication Platform
🔐 Authentication: OAuth, SSO (SAML), MFA, SCIM, Magic links
👥 Roles: Workspace Owner, Admin, Member, Guest, Limited Member
📊 Dashboard: Message threads, channels, DMs, search, notifications, integrations
📱 Features: Real-time sync File sharing Bot API App ecosystem
🔗 Reference: slack.com

Asana

Work Management Platform
🔐 Authentication: OAuth, SAML SSO, Domain verification, IP whitelisting
👥 Roles: Owner, Admin, Editor, Commenter, Guest, Viewer
📊 Dashboard: Project views, timeline, calendar, workload, dashboards, portfolio
📱 Features: Template library Dependencies Automation rules Reporting
🔗 Reference: asana.com

Google Workspace

Productivity & Collaboration Suite
🔐 Authentication: OAuth 2.0, 2-Step verification, SAML, SCIM, Security keys
👥 Roles: Super Admin, User Admin, Group Admin, Org Manager, User
